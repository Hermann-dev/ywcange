== Description ==
Pixi is A Creative Multipurpose WordPress Theme for Creatives that will showcase your for Agencies, Creative, Freelancers, business, Portfolio, app or blogging (plus much more).


== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Upload your theme files pixi.zip
3. Click on the 'Activate' button to use your new theme right away.
4. Go to theme documentation for detailed instructions.
